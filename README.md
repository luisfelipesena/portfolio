# Luis Felipe Sena — Portfolio

Personal portfolio site for **Luis Felipe Sena**, Senior Full-Stack Engineer.

Single-file, dependency-free static site: hand-coded HTML, CSS, and vanilla JS. Dark, motion-driven design with scroll reveals, a cursor spotlight, and a scroll-spy sidebar.

**Live:** https://luisfelipesena.surge.sh

## Stack
- Plain HTML + CSS (custom properties, grid, IntersectionObserver)
- Google Fonts: Space Grotesk, Inter, JetBrains Mono
- No build step, no framework. `index.html` is the whole site.

## Local preview
```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy
```bash
npx surge . luisfelipesena.surge.sh
```
